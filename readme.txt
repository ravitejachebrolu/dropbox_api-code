The dropbox_script.py file contains the code to execute few things:
1) It is used to upload a file and it can be renamed or can be as it is.
2) to check what all are present in your dropbox.returns a dictionary of all the available data
3) to download the files present in the dropbox


The program can be modified ,if you want it to be app  specific

1) changing the app permissions to all types and changing the ACCESSTYPE = "dropbox" can help you
2) to be app specific it should be ACCESSTYPE = "app_folder"
